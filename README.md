## Installation & Start

In the project directory, you can run the following to install dependencies and run app:

```
yarn install
yarn start
```

## Unit Testing

```
yarn test
...
press 'a' for all tests to run following prompts
```

## Integration Testing

```
yarn cypress

or with browser

yarn cypress:browser
```
